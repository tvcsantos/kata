# kata

One tool to configure all your AI harnesses / agents.

Write instructions, MCP servers, prompts, skills, and subagents once in `.kata/`; compile them into every tool's native format (CLAUDE.md, AGENTS.md, .mcp.json, .cursor/, .gemini/, opencode.json, .vscode/, .github/).

```sh
npx @katahq/cli init
npx @katahq/cli plan
npx @katahq/cli apply
```
