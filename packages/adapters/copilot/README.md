# @katahq/adapter-copilot

kata adapter for GitHub Copilot CLI (.github/copilot-instructions.md, .mcp.json, .github/agents, .github/skills).

Built-in adapter shipped with [kata](https://github.com/tvcsantos/kata); you don't normally install it directly.
